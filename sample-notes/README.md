# WriteLess 示例笔记库

> 这是 WriteLess 笔记应用的示例文档库，用于测试文件树导航和 Markdown 渲染功能。

---

## 目录结构

```
sample-notes/

│   └── 设计规范/            # UI/UX 规范

│   ├── 2024-03/            # 3月笔记

│   ├── 前端开发/           # 前端技术

```

## 文档说明

### WriteLess文档/
项目的完整产品和技术文档，包括：
- 创意说明和需求洞察
- 竞品分析
- 产品内核定义
- 技术方案和开发路线图
- 商业模式和增长策略
- 壁垒分析

### 日常笔记/
按月份组织的工作日志，包含待办事项、工作进展、问题记录。

### 学习笔记/
技术学习笔记，包括：
- CodeMirror 6 使用指南
- React 性能优化
- RAG vs 分层索引对比
- Prompt 工程实践

### 项目资料/
项目管理相关文档，包括技术选型决策和里程碑计划。

---

## 测试要点

1. **目录树渲染**
   - 多级目录正确显示
   - 展开/折叠功能正常
   - 文件图标正确

2. **Markdown 渲染**
   - 标题层级正确
   - 列表嵌套显示
   - 代码块高亮
   - 待办事项复选框

3. **Live Preview**
   - 光标不在时语法隐藏
   - 光标进入时语法显示
   - 平滑过渡效果
